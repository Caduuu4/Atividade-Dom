# Atividade DOM

Projeto desenvolvido para praticar conceitos básicos de manipulação do DOM utilizando HTML, CSS e JavaScript.

## Funcionalidades

- Contador de cliques com incremento e decremento
- Alerta quando o contador chega em zero
- Adição de texto dinâmico ao pressionar Enter
- Contador de caracteres sem espaços
- Adição de listas ordenadas ou não ordenadas

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
